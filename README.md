# Jarvis

Мой голосовой помощник для Windows - что-то вроде домашнего «Джарвиса», только
по-русски и на моём компе. Говоришь "Джарвис", он слушает, отвечает голосом и
делает, что просят: открывает программы и сайты, включает музыку, выполняет
команды в терминале, запускает Python, ведёт заметки, список дел и таймеры

Писал под себя, поэтому всё на русском и заточено под Windows

## Что понадобится

- Windows 10 или 11
- Python 3.10
- микрофон
- ключ [OpenRouter] (https://openrouter.ai) - на нём крутится облачная модель

## Установка

```powershell
git clone https://github.com/naturalniyaboba2077/JARVIS-HELPER.git
cd JARVIS-HELPER
python -m pip install -r requirements.txt
copy jarvis_config.example.json jarvis_config.json
```

Дальше открываешь `jarvis_config.json` и вставляешь свой ключ в поле
`OPENROUTER_API_KEY`. Сам файл с ключом в гит не уходит — он в `.gitignore`

## Запуск

```powershell
python jarvis.py
```

Говоришь «Джарвис», ждёшь "Слушаю, сэр" и даёшь команду

## Если нет видеокарты и не хочется ставить модели

Чтобы просто попробовать и чтобы всё работало через облако, поменяй в
`jarvis_config.json` три строчки:

```json
"JARVIS_LLM": "cloud",
"STT_ENGINE": "google",
"TTS_ENGINE": "edge"
```

Тогда ничего локально ставить не надо. А если захочется быстрее и без интернета -
можно докинуть Ollama (`ollama pull qwen2.5:3b`), faster-whisper на видеокарте для
распознавания и Piper для голоса. Как это включить, расписал в
[SETUP_INSTRUCTIONS.md](SETUP_INSTRUCTIONS.md)

Если что-то менял в коде, стоит прогнать:

```powershell

python test_safety.py
```

## Из чего всё состоит

- `jarvis.py` — тут вообще вся программа, одним файлом
- `overlay.py` — полоски-визуализация голоса по краю экрана (отдельный процесс)
- `ui/` — само окно
- `pc_apps.txt` — список установленных программ, по нему работает "открой …"
- `install_shortcut.ps1` — делает ярлык на рабочем столе
